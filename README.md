# 📘 Atividade de JavaScript – Vetores (Arrays)

Este trabalho teve como objetivo o aprendizado da utilização de **vetores (arrays) em JavaScript**, desenvolvendo exercícios práticos envolvendo armazenamento de dados, estruturas de repetição, condicionais e interação com o usuário por meio de entradas via `prompt`.

---

## 1) Soma dos números pares ➕

O programa solicita ao usuário a entrada de 10 números inteiros, armazenando-os em um vetor.  
Em seguida, realiza o cálculo da soma apenas dos números pares presentes no vetor.

---

## 2) Vetor com números aleatórios 🎲

O programa preenche automaticamente um vetor com 7 números gerados aleatoriamente pelo computador.  
Após isso, os valores gerados são exibidos ao usuário.

---

## 3) Contagem de ocorrências 🔍

O usuário insere 10 números inteiros que são armazenados em um vetor.  
Depois, o programa solicita um número adicional e verifica quantas vezes esse número aparece dentro do vetor.

---

## 4) Remoção de valores duplicados 🚫

O usuário digita 10 números inteiros que são armazenados em um vetor.  
Em seguida, é criado um segundo vetor contendo apenas os valores únicos, removendo as duplicatas.  
Por fim, são exibidos o vetor original e o vetor sem repetições.

---

## 💡 Conceitos Praticados

- Estrutura de repetição `for`
- Vetores (arrays)
- Estruturas condicionais (`if`)
- Operadores lógicos e matemáticos
- Entrada de dados com `prompt`
- Saída de dados com `console.log`
- Manipulação manual de índices de vetores

---

## 👨‍💻 Conclusão

Com esta atividade, foi possível compreender melhor o funcionamento dos vetores em JavaScript, praticando o armazenamento, leitura, comparação e manipulação de dados de forma manual, fortalecendo a lógica de programação.
